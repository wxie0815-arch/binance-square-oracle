## 中文特有改写规则

### R1：打破排比，保留最锐利的一句
```
❌ 改变了止损习惯，改变了仓位管理，改变了情绪控制
✅ 就改了一件事：止损单提前设好，不设不开
```

### R2：数字必须具体（禁止模糊）
```
❌ 大幅下跌 / 明显上涨 / 显著增长
✅ 跌了3.8% / 涨了140% / 成交量翻了2.3倍
```

### R3：加时间锚点
```
❌ 最近行情不好
✅ 今天早上8点，$BTC 跌破85000
```

### R4：结尾留问题不留升华
```
❌ 愿每位投资者都能找到属于自己的节奏
✅ 你今天账户什么情况？
```

### R5：允许不完整句
```
✅ 说实话……我也不确定。
✅ 反正今天没爆仓。够了。
✅ 三个月前同样的跌幅，我爆了。
```

### R6：代币格式（广场强制）
- 所有代币前后加空格：` $BTC ` ` $ETH ` ` $XAU ` ` $XAG `
- 链接到币安永续合约

## 币安广场爆款5步结构

```
① 钩子（1句）：数字冲击 / 反差 / 自我暴露
② 背景（2-3句）：具体情境 + 时间锚点
③ 转折（1句）：出人意料的行动/发现
④ 结果（数字/可验证）
⑤ 开放结尾（问句）：引发评论
```

### 爆款开头公式（按场景）
| 场景 | 公式 |
|------|------|
| 跌市 | `$BTC 今天-X%，我[具体动作]` |
| 涨市 | `[代币]刚刚突破[价格]，我[反应]` |
| 洞察 | `一个很多人没注意到的数据：[具体数字]` |
| 反思 | `说个可能得罪人的：[观点]` |
| 操作 | `今天我在[价格]做了[操作]，理由只有一个` |

### 爆款结尾公式
| 类型 | 示例 |
|------|------|
| 问句 | `你今天还好吗？` |
| 留白 | `反正今天没爆仓。够了。` |
| 悬念 | `明天收盘我来更新结果。` |

## 无邪专属词汇表

`欢迎关注` `建议收藏` `干货满满` `未来可期` `前景广阔`
